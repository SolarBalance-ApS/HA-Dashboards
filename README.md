Få at få din SoBa enhed tilsluttet dit Home Assistant(HA), så skal understøttelsen installeres SoBa enheden. Kontakt os på mail, så opdaterer vi softwaren på SoBa One og sender dig den krypteringsnøgle (encryption key) der er nødvendig får at få enheden meldt ind i HA.

Vi har lavet to videoer omkring opstart som du kan se her: https://www.youtube.com/watch?v=RW3uHra71j8&list=PLn4gQYo5zRTDTZEFA2rqg88gZllYbbf8Q
